# Repository Guidance

This repository develops a reproducible benchmark for Vertex Palace.

Use Vertex Palace before repository work. Check status, read the entrance
pitfall board, route the task, and build a minimal context pack before broad
inspection. Keep generated benchmark runs under `.benchmark-runs/`; never
commit transcripts because they may contain local paths or session metadata.

The Control arm generated by this project is an explicit experiment. Its
instruction not to use Vertex Palace applies only inside that disposable arm.
